# Security Policy

## Supported Versions

We currently provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

We take the security of the GOAT Arena project seriously. If you believe you have found a security vulnerability, please report it to us responsibly.

**Please do not report security vulnerabilities via public GitHub issues.**

### Reporting Process

1.  **Email**: Send a detailed report to `security@goatarena.example` (or the project maintainer's primary contact).
2.  **Details to Include**:
    *   Description of the vulnerability.
    *   Steps to reproduce.
    *   Potential impact.
    *   Any suggested fixes.

### Response Timeline

*   **Acknowledgment**: We will acknowledge receipt of your report within 48 hours.
*   **Status Update**: We will provide a status update on the fix within 7 business days.
*   **Disclosure**: We will coordinate the disclosure timeline with you to ensure users have time to update before public disclosure.

Thank you for helping keep GOAT Arena secure!
